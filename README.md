# ahr_antagonists_qsar_supplementary_data
Supplementary materials for QSAR modeling of Aryl Hydrocarbon Receptor antagonists, including curated datasets and analysis outputs.

This repository contains supplementary materials associated with research on Aryl Hydrocarbon Receptor (AhR) ligands.

## Contents

### Data
- QSAR classification dataset (Table S1)
- validation dataset (Table S2)
- classification threshold data (Table S3)

### Documents
- model results and analysis
- fingerprint analysis
- Tanimoto similarity analysis

### Figures
- Murcko scaffold outlier analysis

## Notes
- Data were curated from patents.
- Source information is provided within each dataset.
